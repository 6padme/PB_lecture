온라인 로또 구매 방법 완벽 가이드 — 동행복권 앱·사이트 이용법

2026년 4월 15일 · 읽는 시간 약 7분 2026.04 업데이트

판매점을 직접 방문하지 않아도 스마트폰 앱이나 PC 웹사이트에서 로또를 구매할 수 있습니다. 동행복권이 운영하는 공식 온라인 서비스를 통하면 집에서 편리하게 로또를 구매하고, 당첨 결과까지 자동으로 확인할 수 있습니다. 이 글에서는 회원가입부터 구매, 당첨금 수령까지 전 과정을 단계별로 안내합니다.

1. 온라인 로또 구매의 장단점

구분장점단점
온라인 구매24시간 구매 가능, 당첨 자동 확인, 분실 걱정 없음주 10만원 구매 한도, 예치금 충전 필요
오프라인 구매한도 제한 없음, 현금 즉시 결제영업시간 제한, 복권 분실 위험

온라인 구매의 가장 큰 장점은 당첨 복권을 잃어버릴 걱정이 없다는 점입니다. 실제로 매년 수십억 원 규모의 당첨 복권이 기한 내 신청되지 않는데, 그 상당수가 복권 분실이나 기한 초과 때문입니다. 온라인 구매 시에는 당첨금이 예치금으로 자동 적립됩니다.

💡 온라인 구매 가능 시간
로또 6/45는 매주 월요일 오전 6시 ~ 토요일 오후 8시까지 구매 가능합니다. 토요일 오후 8시 이후에는 추첨 준비로 구매가 마감됩니다.

2. 동행복권 회원가입 방법

온라인으로 로또를 구매하려면 동행복권 공식 사이트 또는 앱에서 본인 인증 후 회원가입이 필요합니다.

① PC(웹사이트) 가입

  1. 동행복권 공식 홈페이지 접속
  2. 상단 메뉴 [회원가입] 클릭
  3. 약관 동의 후 본인 인증 (휴대폰 또는 공동인증서)
  4. 아이디·비밀번호 설정 및 계좌 등록
  5. 가입 완료 — 예치금 충전 후 구매 가능

② 모바일 앱 가입

  1. 앱스토어 또는 구글플레이에서 '동행복권' 검색·설치
  2. 앱 실행 후 [회원가입] 탭 선택
  3. 휴대폰 본인 인증 진행
  4. 계좌 인증 후 가입 완료
  5. 간편 결제(카카오페이, 네이버페이 등) 연동 가능
⚠️ 가입 자격 제한
만 19세 미만, 금융 거래 제한자, 사행산업 관련 종사자 등은 가입이 제한됩니다. 1인 1계정만 허용되며 중복 가입 시 이용이 정지됩니다.

3. 예치금 충전 방법

온라인 로또 구매는 선불 방식으로, 미리 예치금을 충전해야 합니다. 충전 방법은 다음과 같습니다.

충전 방법최소 금액처리 시간
계좌이체 (가상계좌)1,000원즉시
신용·체크카드5,000원즉시
간편결제 (카카오페이 등)1,000원즉시

예치금 잔액은 언제든지 등록된 본인 명의 계좌로 출금할 수 있습니다. 단, 본인 명의 계좌로만 출금이 가능하므로 처음 등록 시 정확한 계좌 정보를 입력해야 합니다.

4. 로또 온라인 구매 방법 (단계별)

자동번호 구매

  1. [로또 6/45] 메뉴 선택
  2. 구매 게임 수 선택 (1게임~5게임)
  3. [자동] 선택 후 [확인] 클릭
  4. 예치금에서 금액 차감 후 구매 완료
  5. 마이페이지에서 구매 내역 확인 가능

수동번호 구매

  1. [로또 6/45] 메뉴에서 [수동] 선택
  2. 1~45 중 원하는 숫자 6개 직접 선택
  3. 게임별로 번호 입력 후 [구매] 클릭
  4. 자동과 수동을 조합한 혼합 구매도 가능

5. 구매 한도 및 제한 사항

구분한도비고
1회 구매 한도최대 5게임 (5,000원)1회 구매 기준
주간 구매 한도10만원월~토 기준
연간 구매 한도별도 제한 없음주 10만원 내에서 누적
예치금 최대 보유200만원초과 시 충전 불가

주 10만원 한도는 사행성 조장 방지를 위한 법적 규제입니다. 오프라인 판매점에는 이런 한도가 없으므로, 대량 구매를 원한다면 오프라인을 이용해야 합니다.

6. 당첨 확인 및 당첨금 수령

온라인 구매의 가장 큰 장점은 당첨 결과가 자동으로 확인되고 당첨금이 자동 적립된다는 점입니다.

당첨금 자동 적립 기준

등수처리 방식수령 방법
5등 (5,000원)예치금 자동 적립예치금 출금
4등 (5만원)예치금 자동 적립예치금 출금
3등 (약 150만원)예치금 자동 적립예치금 출금
2등 (약 6천만원)온라인 청구 후 계좌 지급동행복권 고객센터 연락
1등 (수십억원)동행복권 본사 방문 필요신분증 지참, 은행 계좌 개설 필요
💡 당첨금 청구 기한
추첨일로부터 1년 이내에 청구하지 않으면 당첨금이 소멸됩니다. 온라인 구매 시 마이페이지에서 당첨 내역이 자동 표시되지만, 금액이 큰 경우 기한 안에 반드시 청구 절차를 진행하세요.

7. 자주 묻는 질문 (FAQ)

질문답변
구매 후 취소할 수 있나요?구매 완료 후에는 취소·환불이 불가합니다.
여러 계정을 만들어도 되나요?1인 1계정 원칙이며, 중복 적발 시 계정 정지됩니다.
공동 구매를 온라인으로 할 수 있나요?개인 계정으로만 구매 가능하므로, 공동 구매는 별도 약정이 필요합니다.
해외에서 구매할 수 있나요?국내 본인 인증이 필요하므로 해외에서는 구매가 어렵습니다.
앱이 없어도 구매되나요?PC 웹사이트로도 구매 가능합니다.

8. 보안 주의사항

  • 동행복권 공식 앱·사이트만 이용 — 유사 사기 사이트 주의
  • 로그인 비밀번호는 주기적으로 변경 권장
  • 예치금 잔액이 남아 있다면 미사용 예치금은 즉시 출금
  • 카카오·네이버 간편 로그인 사용 시 해당 계정 보안 강화 필수
  • 타인 계정 대리 구매는 불법이며 당첨금 수령이 불가합니다

결론 — 편리하지만 한도 내에서 즐기기

동행복권 온라인 서비스는 복권 분실 걱정 없이 언제 어디서나 구매할 수 있는 편리한 방법입니다. 주 10만원의 구매 한도는 과도한 지출을 방지하는 안전장치이기도 합니다. 복권은 어디까지나 소액의 오락으로, 정해진 예산 안에서 즐기는 것이 가장 현명한 방법입니다.
